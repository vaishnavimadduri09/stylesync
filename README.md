# 👗 OOTD Intelligence Platform

## Live App
🌐 [Click here to use the app!](https://ootd-intelligence-platform.streamlit.app/)

## Problem
Every morning people struggle with "what do I wear today?" — they don't know 
the weather, don't know what's trending and don't know what suits the occasion!

## Solution
An AI powered web app that suggests the perfect outfit from YOUR wardrobe 
based on live weather, occasion and global fashion trends!

## Features
- 👗 Upload your clothes and get personalized outfit suggestions
- 🌤️ Live weather integration for any city in the world
- 🤖 AI powered styling advice using Claude AI
- 📊 Global fashion analytics dashboard
- 🎨 Trending colors and styles by country

## Pipeline Architecture
User Uploads Clothes
↓

Live Weather API

↓

Claude AI Analysis
↓
Personalized Outfit Suggestion
↓
Fashion Analytics Dashboard
↓
Deployed Live on Streamlit Cloud

## Tech Stack
- Python
- Streamlit
- Anthropic Claude AI
- OpenWeather API
- Plotly
- Git & GitHub

## How to Use
1. Go to the live app
2. Enter your city
3. Select your occasion
4. Upload photos of your clothes
5. Click "Get My Outfit Suggestion!"

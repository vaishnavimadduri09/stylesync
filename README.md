# 👗 StyleSync

## Live App
🌐 [Click here to use StyleSync!](https://stylesync-app.streamlit.app/)

## Problem
Every morning people struggle with "what do I wear today?" — they don't know 
the weather, what's trending or what suits the occasion!

## Solution
StyleSync is a fully deployed AI powered web app that suggests the perfect 
outfit from YOUR wardrobe based on live weather, occasion and global fashion 
trends. Users can create accounts, save their wardrobe and track their 
outfit history!

## Features
- 👤 User authentication — create account and login
- 👗 Upload your clothes and get personalized outfit suggestions
- 🌤️ Live weather integration for any city in the world
- 🤖 AI powered styling advice using Claude AI
- 📊 Global fashion analytics dashboard
- 🎨 Trending colors and styles by country
- 📝 Outfit history — save and revisit past suggestions

## Pipeline Architecture

| Step | Description |
|---|---|
| 1️⃣ User Login | Secure authentication with encrypted passwords |
| 2️⃣ Upload Clothes | Upload photos of your wardrobe |
| 3️⃣ Live Weather API | Fetches real time weather for your city |
| 4️⃣ Claude AI Analysis | Analyzes your clothing items |
| 5️⃣ Outfit Suggestion | Personalized recommendation generated |
| 6️⃣ Save to History | Outfit saved to your personal history |
| 7️⃣ Fashion Dashboard | Global trend analytics displayed |
| 8️⃣ Deployed Live | Available on Streamlit Cloud |

## Tech Stack
- Python
- Streamlit

## How to Use
1. Go to the live app
2. Create an account or login
3. Enter your city
4. Select your occasion
5. Upload photos of your clothes
6. Click "Get My Outfit Suggestion!"
7. View your outfit history anytime!
